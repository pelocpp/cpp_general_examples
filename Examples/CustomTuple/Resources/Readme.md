# Realisierung der Standardklasse `std::tuple<>`

In zwei Realisierungen wird eine vereinfachte Realisierung der Standardklasse `std::tuple<>` betrachtet.

Die erste Realisierung ist in dem Buch *"C++ Templates: The Complete Guide (2nd Edition)"*
von David Vandevoorde, Nicolai M. Josuttis und Douglas Gregor näher beschrieben
(siehe [hier](http://www.josuttis.de/tmplbook))

Ein anderer Ansatz wird von Eli Bendersky in dem Artikel
[Variadic templates in C++](https://eli.thegreenplace.net/2014/variadic-templates-in-c)
eingeschlagen. 

Eine weitere Realisierung wird in dem Blog von Vishal Chovatiya vorgestellt:
[Variadic Template C++: Implementing Unsophisticated Tuple](http://www.vishalchovatiya.com/variadic-template-cpp-implementing-unsophisticated-tuple)

Alle drei Beispiele dienen einzig und alleine dem Zweck, 
ein tieferes Verständnis für variadische Templates als auch die Klasse `std::tuple<>` zu erlangen.
